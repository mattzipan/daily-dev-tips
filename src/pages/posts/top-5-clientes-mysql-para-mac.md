Cuando se trata de bases de datos, MySQL es un jugador importante y ampliamente utilizado.
Hoy veremos algunos clientes de MySQL que podemos usar en Mac para ver estas bases de datos.

> TL;DR: Mi favorito personal para Mac es [TablePlus](https://tableplus.com/). Ofrece una variante gratuita que es suficiente.

## 1 TablePlus

TablePlus es el cliente de MySQL que estoy usando actualmente.
Todavía estoy en el plan gratuito, que solo te permite tener dos pestañas abiertas a la vez, lo cual es suficiente para mí.

<img src="https://daily-dev-tips.com/cdn-cgi/imagedelivery/Bki7Af2hq0JKVFw1XYYMQg/c252cd80-8ad7-4689-502c-11830c9c5d00/article" alt="TablePlus MySQL client for Mac" width="750" height="500" />

#### PROS

- Interfaz clara
- Edita rápidamente varias filas a la vez
- Consulta RAW siempre visible

#### CONTRAS

- Solo dos pestañas en la versión gratuita
- No hay clic directo para consultas enlazadas

[Descargar TablePlus](https://tableplus.com/)

## 2 phpMyAdmin

phpMyAdmin es otro recurso excelente para ver tus bases de datos MySQL. Es muy bueno ya que la mayoría de los servidores vendrán con phpMyAdmin instalado. Lo que lo convierte en una buena interfaz universal.

<img src="https://daily-dev-tips.com/cdn-cgi/imagedelivery/Bki7Af2hq0JKVFw1XYYMQg/f544b123-1366-498f-a133-375777a57900/article" alt="phpMyAdmin MySQL client for Mac" width="750" height="500" />

#### PROS

- Cercano a la interfaz de tu servidor para MySQL
- Fácil de instalar
- Basado en la web

#### CONTRAS

- No hay resaltado de sintaxis
- La importación/exportación depende de la web, lo que los hace influenciados por la configuración del servidor

[Descargar phpMyAdmin](https://www.phpmyadmin.net/)

## 3 DBeaver

DBeaver es una herramienta de base de datos gratuita y de código abierto. Está construida alrededor de la usabilidad.

Esta herramienta es completa y tiene muchas características interesantes. Sin embargo, para mí, se siente un poco demasiado.

<img src="https://daily-dev-tips.com/cdn-cgi/imagedelivery/Bki7Af2hq0JKVFw1XYYMQg/26bbbb6c-dc2f-4338-9e7c-fa7c5c442d00/article" alt="DBeaver MySQL client for Mac" width="750" height="500" />

#### PROS

- Lleno de herramientas útiles
- ¡Diagramas ER!
- Clic directo en consultas enlazadas

#### CONTRAS

- Un poco torpe para mi gusto

[Descargar DBeaver](https://dbeaver.io/download/)

## 4 Querious

Interfaz muy similar a Mac y clara para entender cómo funciona y qué hace.
Realmente me gusta la herramienta, sin embargo, debería haber sido una herramienta gratuita.

<img src="https://daily-dev-tips.com/cdn-cgi/imagedelivery/Bki7Af2hq0JKVFw1XYYMQg/5564d7a6-2caa-4324-8454-f4a6bd310000/article" alt="Querious MySQL client for Mac" width="750" height="500" />

#### PROS

- Diseñado para Mac
- Muy buena interfaz para MySQL

#### CONTRAS

- 25\$

[Descargar Querious](https://www.araelium.com/querious)

## 5 Sequel Pro

Sequel Pro solía ser mi cliente de MySQL favorito para Mac, sin embargo, últimamente parece tener bastantes fallos. La razón por la que cambié a TablePlus.

La herramienta en sí tiene una de las mejores interfaces que he visto para MySQL.

<img src="https://daily-dev-tips.com/cdn-cgi/imagedelivery/Bki7Af2hq0JKVFw1XYYMQg/d2a3a90d-1fdb-4632-3c55-91b2376dbe00/article" alt="Sequel Pro MySQL client for Mac" width="750" height="500" />

#### PROS

- Interfaz muy clara
- Fácil de entender y usar

#### CONTRAS

- Se cuelga con frecuencia, lamentablemente en las últimas versiones

[Descargar Sequel Pro](https://www.sequelpro.com/)
